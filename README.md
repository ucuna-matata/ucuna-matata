## 📦 Installation

### Local

```bash
# Clone repository
git clone https://github.com/partum55/ucuna-matata.git
cd ucuna-matata

# Install dependencies
pip install -r requirements.txt

# Run
python app.py
```

Open browser: `http://localhost:5000`

### Docker

```bash
docker-compose up
```
